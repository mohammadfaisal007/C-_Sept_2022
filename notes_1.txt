Value Data Types
- Numeric
    - Integer
        - sbyte - signed byte       - 1 byte (8 bit)
        - byte  - unsigned byte     - 1 byte (8 bit)
        - short - signed            - 2 byte (16 bit)
        - ushort - unsigned         - 2 byte (16 bit)
        - int   - signed            - 4 bytes (32 bit)
        - uint  - unsigned          - 4 bytes (32 bit)
        - long  - signed            - 8 bytes (64 bit)
        - ulong - unsigned          - 8 bytes (64 bit)
    - Floating Point
        - float     - 4 bytes
        - double    - 8 bytes
        - decimal   - 16 bytes

- Character
    - char - 2 byte - 16 bit
